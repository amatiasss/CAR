Ajuda para o cadastramento de e-mails da equipe

responsável pela análise dos serviços disponibilizados

 

    No cadastramento de e-mails temos os seguintes campos:

  •     Nome do Responsável: este campo deve ser preenchido com o nome do responsável que irá analisar um dos serviços disponibilizados pelo sistema.

  •     E-mail para cadastrar: este campo deve ser preenchido com o e-mail do responsável que irá analisar um dos serviços disponibilizados pelo sistema.

  •     Setor responsável: neste campo será escolhido qual o setor do serviço disponibilizado no sistema.

    Após o preenchimento dos campos, deve-se cadastrar o novo e-mail clicando no botão cadastrar. Caso não queira cadastrar, cancela-se o cadastramento no botão cancelar.
