Ajuda para o preenchimento do

formulário de reclamação

 

Existem quatro campos que devem ser preenchidos obrigatoriamente. São eles:

 

  • Nome: Este campo deve ser preenchido com o nome do reclamante.
  • E-mail para contato: Este campo deve ser preenchido com o e-mail do reclamante, para que este seja avisado sobre o andamento da reclamação.
  • Campo para marcar o tipo de reclamação: Neste campo, marca-se o tipo da reclamação que se deseja fazer. Os tipos são:

 

    • Equipamento – quando a reclamação for referente a algum equipamento, como computadores, data show, cadeiras, etc;
    • Rede – quando a reclamação for referente à rede;
    • Pessoal de apoio - quando a reclamação for referente aos funcionários e alunos que monitoram os serviços do bloco REENGE;
    • Outros - quando a reclamação for referente a algum tópico que não seja abordado pelos tópicos acima.

 

  • Campo para preenchimento da reclamação: Este campo é preenchido pelo reclamante de forma a abordar com mais detalhes o tipo da reclamação que está sendo feita.

 

Um único campo que não tem obrigatoriedade de preenchimento é o campo do ramal, este deve ser preenchido com o ramal do reclamante, caso tenha um, com o intuito de informar sobre o andamento da reclamação.

Após o preenchimento dos campos, deve-se submeter à solicitação clicando no botão Submeter.
