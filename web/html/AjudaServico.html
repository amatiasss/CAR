Ajuda para o preenchimento do

formulário de solicitação do serviço

 

Existem campos que devem ser preenchidos obrigatoriamente. São eles:

 

  • Nome: Este campo deve ser preenchido com o nome do solicitante.
  • E-mail para contato: Este campo deve ser preenchido com o e-mail do solicitante, para que este seja avisado sobre o andamento da solicitação.
  • Campo para marcar o tipo de serviço solicitado: Neste campo, marca-se o tipo do serviço que se deseja solicitar. Ao marcar um tipo, serão visualizados os campos referentes a cada tipo de serviço para o preenchimento dos mesmos. Os serviços são:
    • Instalação de software – quando este tipo de serviço for marcado, será visualizado um campo  não obrigatório, para que seja fornecido um link para download do software, no caso de ele ser livre;
    • Abertura de conta – quando este tipo de serviço for marcado, serão visualizados dois campos obrigatórios, um para abordagem dos motivos da reabertura da nova conta e outro, pedindo uma sugestão para o nome da conta;
    • Manutenção de conta – quando este tipo de serviço for marcado, serão visualizados dois campos obrigatórios, um pedindo para que seja informada a manutenção desejada e outro pedindo uma sugestão para o nome da conta;
    • Manutenção de rede ou máquina - quando este tipo de serviço for marcado, serão visualizados dois campos obrigatórios, um para que seja marcado se a manutenção é de rede ou de máquina, e outro, pedindo para que seja informada a natureza do problema.

     

Um campo que não tem obrigatoriedade de preenchimento é o campo do ramal, este deve ser preenchido com o ramal do solicitante, caso tenha um, com o intuito de informar sobre o andamento da solicitação.

Após o preenchimento dos campos, deve-se submeter à solicitação clicando no botão Submeter.
