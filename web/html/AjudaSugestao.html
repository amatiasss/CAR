Ajuda para o preenchimento do

formulário de sugestão

 

 

Existem três campos que devem ser preenchidos obrigatoriamente. São eles:

 

  • Nome: Este campo deve ser preenchido com o nome de quem irá fazer a sugestão.
  • E-mail para contato: Este campo deve ser preenchido com o e-mail de quem irá fazer a sugestão.
  • Campo para preenchimento da sugestão: Este campo é preenchido com a sugestão que será fornecida.

 

Um único campo que não tem obrigatoriedade de preenchimento é o campo do ramal, este deve ser preenchido com o ramal de quem irá fazer a sugestão, caso tenha um.

Após o preenchimento dos campos, deve-se submeter à sugestão clicando no botão Submeter.
